# Focus with Bruno

## Installing

```bash
# From Source
git clone https://github.com/bokhariLovesYou/focus-with-bruno-client.git
cd focuswithbruno-client
```

then install dependencies

```bash
  npm install
```

## Extra options

```bash
# Run a production build, locally
gatsby build

# Serve a production build, locally
gatsby serve
```
